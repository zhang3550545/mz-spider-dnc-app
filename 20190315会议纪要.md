### 3.python命名规范

记住一个，文件名，方法名，变量名，一律小写，多个单词使用下划线分割

```
# 文件名命令规范
common.py
read_file.py

# 方法名命名规范
def print():
	pass

def read_file():
	pass

# 变量名命名规范

num = 0

arrays=[1,2,3]

name = "曹操"
```
